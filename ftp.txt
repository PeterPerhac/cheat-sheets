# FTP - from the command line

**Connect** like so: ftp user@host

## Remote commands

 - cd
 - ls
 - mkdir
 - pwd

The above work as usual, but take effect on the remote server

## Local commands

 - lcd
    local cd
 - lpwd
    local pwd
 - !ls
    execute (!) the ls command (locally)

## Download
    
 - get
    single file download
 - mget
    multiple get
        mget *.jpg

## Upload

 - put
    single file upload
 - mput
    multiple file upload

## Quit

 - bye
 - exit
 - quit
    to say goodbye
